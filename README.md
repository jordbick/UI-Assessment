# UI Assessment

A simple React UI component library and assessment project.

## Main Features

- **Reusable rating components**
  - Average star displays
  - Rating bars
  - Accessible components with ARIA support
- **Test coverage with Jest & React Testing Library**
- **Component previews and documentation in Storybook**
- **Flexible theming support using `styled-components`**

## Installation

- Clone repo
- Run either:
  - npm run dev
  - npm run storybook
- npm run test to run unit tests

## Storybook

Use storybook to preview components, adjust props via Controls, and check accessibility using the A11y addon in the Storybook UI.
